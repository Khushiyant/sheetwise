Quick Start

Three simple steps to compress your spreadsheets

1

Install

pip install sheetwise
2

Import & Load Data

from sheetwise import SpreadsheetLLM import pandas as pd df = pd.read_csv("your_data.csv")
3

Compress & Get Results

sllm = SpreadsheetLLM() compressed = sllm.compress_and_encode_for_llm(df) stats = sllm.get_encoding_stats(df) print(f"Reduced by {stats['compression_ratio']:.1f}x")
